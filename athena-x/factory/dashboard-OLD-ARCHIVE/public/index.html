🧙‍♂️ SiteType Wizard - Gebruiksgids

🎯 Wat is een SiteType?

Een SiteType is een blauwdruk voor een type website. Het definieert:

  • 📊 Data structuur - welke informatie er nodig is
  • 🎨 Design systeem - visuele stijl en kleuren
  • 🧩 Parser logica - hoe data wordt verwerkt
  • 📱 Componenten - herbruikbare UI elementen

🔄 Workflow in Athena Management System

1
Data Bron Aanmaken

Maak een nieuw bronproject in de "Data Bronnen" sectie. Upload je data (CSV, Excel, etc.).

2
SiteType Ontwerpen

Gebruik deze wizard om een custom blauwdruk te maken die past bij je data en branche.

3
Site Genereren

Koppel je databron aan de nieuwe SiteType en genereer een complete website.

4
Deployen

Publish de site naar GitHub Pages of andere hosting platforms.

🤖 AI Assistentie

De wizard gebruikt Google Gemini AI voor:

  • Data Structuur - Analyseert je business om relevante data-eenheden te identificeren
  • Design Systeem - Genereert passende kleuren, fonts en stijl voor de branche
  • Parser Instructies - Creëert specifieke regels voor data-extractie

💡 Best Practices

✅ Wees specifiek in je business beschrijving - "luxe advocatenkantoor" werkt beter dan "bedrijf"
✅ Houd data structuren simpel - 2-4 tabellen met 3-6 kolommen per tabel is ideaal
✅ Test je SiteType met een kleine dataset voordat je grootscale gaat gebruiken
✅ Je kunt later altijd componenten aanpassen in de gegenereerde sitetype map

📁 Bestandsstructuur

Een nieuwe SiteType wordt opgeslagen in:

3-sitetypes/[naam]/
  • blueprint/ - JSON configuratie en data definities
  • web/ - React componenten en pagina's
  • parser/ - Data processing logica

🔧 Integratie met Bestaande Tools

Nadat een SiteType is aangemaakt, kun je gebruik maken van:

  • Layout Wizard - Pas de pagina-layouts aan
  • Content Parser - Verwerk brongegevens automatisch
  • Deploy Wizard - Publiceer naar productie
  • Sheet Linker - Koppel Google Sheets voor live data
